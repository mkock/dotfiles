# dotfiles

My dotfiles for NeoVim.

Inspired by [bashbunni's dotfiles](https://github.com/bashbunni/dotfiles/tree/main/.config/nvim).

## Installation process

1. Install NeoVim
2. Put the config files in your `~/.config` directory
3. Run `:PlugInstall` from inside NeoVim
4. For some of the plugins, you'll need Go, Haskell, git, gopls, a nerdy font etc.
